# RepositorioPagesCH61
Repositorio de prueba para pages Java Generation bootcamp CH 61 
JR Coffe — Mini E-commerce con Bootstrap 5

Este proyecto forma parte del curso MEX - JFSD - M61 - 2025.
Consiste en reconstruir la “Plantilla” utilizando Bootstrap 5 para lograr un sitio web responsivo.
La temática elegida es JR Coffe, una cafetería donde se pueden comprar productos de café en grano y molido.

Páginas incluidas

Inicio: Presentación de la marca JR Coffe.

Productos: Catálogo de cafés en grano y molido.

Contacto: Formulario con validación Bootstrap.

Carrito: Carrito funcional con JavaScript y localStorage.

Tecnologías utilizadas

HTML5

Bootstrap 5 (CDN)

JavaScript

LocalStorage para persistencia del carrito

GitHub Pages para despliegue

Funcionalidades principales

Navegación responsiva con Bootstrap.

Catálogo dinámico generado desde JavaScript.

Carrito de compras con:

Agregar productos

Incrementar o disminuir cantidades

Botón para vaciar carrito

Cálculo automático del total

Persistencia mediante localStorage

Formulario de contacto con validación de Bootstrap.
